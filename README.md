# atproto-oauth

Production-focused Agent Skill for implementing AT Protocol OAuth login, discovery, scopes, and security-critical validation correctly.

## Install

```bash
npx skills add pixeline/atproto-oauth
```

## Compatibility

Works with Cursor, Claude Code, Windsurf, Cline, Codex, and other tools that support the skills.sh format.
